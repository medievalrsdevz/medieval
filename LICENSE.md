## 🔧 Troubleshooting Guide

### 🔴 GUI Not Appearing After Execution

**Cause:** Script may have executed before the game fully loaded.
**Fix:** Wait for the game lobby to fully render, then re-execute. Add a 3-second delay in your injection tool's auto-execute settings.

---

### 🔴 Auto Farm Stops After a Few Minutes

**Cause:** Server-side anti-cheat throttle or session timeout.
**Fix:** Enable `safeMode` in your config and lower `farmRadius` to `100`. Also enable the Session Cooldown Manager.

---

### 🔴 ESP Overlays Are Invisible

**Cause:** GPU rendering layer conflict.
**Fix:** Set `espColor` to a high-contrast value like `"#FF4444"` and ensure your graphics settings are not on the lowest preset.

```bash
# Quick fix via config edit
sed -i 's/"espColor": ".*"/"espColor": "#FF4444"/' config.json
```

---

### 🔴 Script Crashes on Mobile

**Cause:** Insufficient RAM or background app interference.
**Fix:** Close all background apps before launching. On Android, enable "High Performance" mode in developer options.

---

### 🔴 Hotkeys Not Responding

**Cause:** Key binding conflict with the game client.
**Fix:** Reassign hotkeys in `config.json` to function keys (`F7`, `F8`, `F9`) that are less likely to conflict.

---

### 🟡 Slow Farm Speed Even With Max Settings

**Cause:** `humanizeDelay` is working correctly — it's intentional.
**Fix:** This is a safety feature, not a bug. You can reduce `delayMin` to `0.15` but do not set it below `0.1` as this increases detection risk.
